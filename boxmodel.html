Basketball

Esta es una tarjeta de ejemplo que utiliza el modelo de cajas y Flexbox para su diseño.

Gastronomía

Cada tarjeta tiene bordes, padding y márgenes definidos para mostrar el box model.

Natación

El contenedor usa Flexbox con justify-content: space-around para distribuir las tarjetas.

Voleibol

Las tarjetas se ajustan automáticamente en múltiples filas gracias a flex-wrap: wrap.
